## Descripción

El tipo de módulo *Estado Multilingüe* muestra un ícono en la Barra de Título para proporcionar acceso a los parámetros multilingües en sitios multilingües.

### Elementos Comunes

Algunos elementos de esta página están cubiertos en artículos de ayuda separados:

* [Barras de herramientas](jdocmanual?article=help/common-elements/toolbars).
* [La pestaña Módulos: Módulo](jdocmanual?article=help/modules/modules-module-tab).
* [La pestaña Módulos: Avanzado](jdocmanual?article=help/modules/modules-advanced-tab).
* [La pestaña Permisos](jdocmanual?article=help/common-elements/edit-permissions).


## Cómo Acceder

- Selecciona **Sistema → Administrar Panel → Módulos del Administrador** desde
  el menú del Administrador. Luego...
  - Para crear un nuevo módulo: selecciona el botón **Nuevo** en la barra de herramientas. Luego...
    - Selecciona el tipo de módulo requerido.
  - Para editar un módulo existente:
    - Encuentra el módulo en la lista de módulos instalados y selecciona el
      enlace del título en la columna **Título**.

## Campos del Formulario

- **Título** El título del módulo. Este es también el título mostrado
  para el módulo dependiendo del campo de formulario *Mostrar Título*.

### Pestaña del Módulo

#### Panel Izquierdo

- **Información** Este módulo muestra el estado de los parámetros multilingües.
