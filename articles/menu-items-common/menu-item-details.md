## Descripción

Todos los elementos del menú tienen un diseño similar, pero algunos de los campos del formulario y algunas de las pestañas cambian de tipo a tipo. Esta página describe el diseño inicial de la pestaña **Detalles** antes de seleccionar un tipo de elemento del menú.

## Cómo Acceder

* Seleccione cualquier **Menú del Sitio** desde el menú de Administrador.
* Seleccione el botón **Nuevo** desde la Barra de Herramientas.

## Campos de Formulario

- **Título del Menú** El título que se mostrará para este elemento del menú.
- **Alias** El nombre interno del ítem. Normalmente, puedes dejarlo
  en blanco y Joomla rellenará un valor predeterminado con el título en minúsculas y
  guiones en lugar de espacios.

### Panel Izquierdo

- **Tipo de Ítem de Menú** El tipo de ítem de menú seleccionado cuando se crea este 
  ítem del menú. Esto puede ser uno de los tipos de ítem de menú estándar o un tipo 
  de ítem de menú proporcionado por una extensión instalada.
- **Ítem de Menú** Este campo aparece después de seleccionar un tipo de ítem de menú. Permite
  seleccionar un menú desde un diálogo emergente para el campo del enlace.
- **Enlace** El enlace generado por el sistema para este ítem de menú. Este campo
  no se puede cambiar y es solo para información.
- **Ventana de Destino** Selecciona desde la lista desplegable.
- **Estilo de Plantilla** Selecciona desde la lista desplegable.

### Panel Derecho

- **Menú** Muestra en qué menú aparecerá el enlace.
- **Ítem Principal** El ítem principal del menú para este ítem del menú. Se utiliza para
  determinar si un ítem de menú es un ítem de nivel superior o un submenú.
  Selecciona *- Sin Principal -* (el valor predeterminado) si este es un ítem de menú de nivel 
  superior. De lo contrario, selecciona el ítem de menú que es el principal de este ítem.
- **Orden** Selecciona *- Primero -* o *- Último -* o el ítem que debería seguir este ítem.
- **Estado** El estado publicado del ítem.
- **Inicio de Publicación** Fecha y hora para iniciar la publicación. Usa este
  campo si deseas ingresar contenido con anticipación y luego publicarlo
  automáticamente en el futuro.
- **Fin de Publicación** Fecha y hora para finalizar la publicación. Usa este
  campo si deseas que el contenido se cambie automáticamente al estado no publicado
  en un momento futuro (por ejemplo, cuando ya no sea aplicable).
- **Página Predeterminada** Si es Sí, este ítem de menú será la página predeterminada o página de inicio
  del sitio. Debe haber exactamente un ítem de menú establecido como la página predeterminada.
- **Acceso** El nivel de acceso de visualización para este ítem.
- **Idioma** Idioma del ítem.
- **Nota** Esto es normalmente para el uso del administrador del sitio (por
  ejemplo, para documentar información sobre este ítem) y no se muestra en
  el Frontend del sitio.
